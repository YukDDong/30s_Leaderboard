Tennis League Dashboard

테니스 리그 결과 입력 및 순위표 관리

팀을 등록하고, 경기 일정을 생성한 뒤, 각 경기 점수만 입력하면 순위표가 자동으로 갱신됩니다. 서버 없이 브라우저와 localStorage만으로 동작합니다.

정적 웹 / GitHub Pages

초보자도 바로 실행하고 배포할 수 있도록 단순한 구조로 만들었습니다.

Overview

진행 현황

Step 1

팀 등록

팀명을 직접 입력해 리그 참가 팀을 추가하세요.

Step 2

등록된 팀 목록

경기 생성 전까지만 팀 삭제가 가능합니다. 경기 생성 후에는 데이터 정합성을 위해 팀 편집이 잠깁니다.

Step 3

경기 생성

팀이 2개 이상일 때 단일 풀리그 경기 목록을 생성할 수 있습니다.

단일 라운드 로빈

각 팀은 다른 모든 팀과 1번씩 경기합니다.

Step 4

경기 목록 및 점수 입력

표에서 경기를 빠르게 훑고, 원하는 행을 눌러 모달에서 점수를 입력하거나 수정하세요.

경기 행을 클릭하거나 입력 버튼을 누르면 모달이 열립니다. 저장 또는 초기화 후 순위표와 진행 현황이 즉시 갱신됩니다.

Step 5

순위표

승점 내림차순, 이후 득실차 내림차순으로 자동 정렬됩니다.

순위 팀명 경기수 승 무 패 득점 실점 득실차 승점

Maintenance

데이터 초기화

팀 목록과 경기 기록을 모두 삭제하고 처음 상태로 되돌립니다.

주의

초기화 후에는 현재 브라우저에 저장된 리그 데이터가 모두 사라집니다.

Match Editor

경기 점수 입력

점수 저장 시 승/무/패와 순위표가 자동으로 갱신됩니다.

미입력
:

점수는 0 이상의 정수만 입력할 수 있습니다.
